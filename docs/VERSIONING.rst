Final Releases
--------------

::
  X(.Y)*

Pre-releases
------------

.. highlight:: html

::
  X.YaN   # Alpha release
  X.YbN   # Beta release
  X.YrcN  # Release Candidate
  X.Y.Z   # For bug-fix releases

Post-releases
-------------

.. highlight:: html

::
  X.YaN.postM   # Post-release of an alpha release
  X.YbN.postM   # Post-release of a beta release
  X.YrcN.postM  # Post-release of a release candidate
